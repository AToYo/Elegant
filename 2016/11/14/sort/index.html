要求：

  • 一个字符串中，每个单词中都包含一个数字，数字的大小决定最后这个单词的所在位置
  • 如果字符串是空的，返回空字符串

Example：

  • “is2 Thi1s T4est 3a” => return “Thi1s is2 3a T4est”

我的代码：

function order(words){

  //将字符串转化为数组

  var arr = words.split(' ');

  //利用map方法遍历数组中的每一个元素

      var num = arr.map(function(item,index){

             for(var i = 0;i < item.length; i ++){
                 //提取每个元素中包含的数字，并返回出去；
                  if(!isNaN(item[i])){
                    return item[i]
                   }
             }
      })

     for(var i = 0; i < arr.length; i ++){

        for(var j = 0; j < num.length - 1 - i;j ++){
        //对数组中的每个元素进行比较，将大的元素赋值给变量temp
           if(num[j] > num[j+1]){

            var temp = num[j];
            //  对元素进行交换
            num[j] = num[j+1];

            num[j+1] = temp; 

            temp = arr[i];

            arr[j] = arr[j+1];

            arr[j+1] = temp;
           }
        }
     }
      return arr.join(' ');
 }

大神代码：

function order(words) {
  return words.split(" ").sort((a, b) => a.replace(/[^\d]/g, "") - b.replace(/[^\d]/g, "")).join(" ")
}

sort()方法：

1. arrayObject.sort(sortby)；参数sortby可选。规定排序顺序。必须是函数。
2. 若 a 小于 b，在排序后的数组中 a 应该出现在 b 之前，则返回一个小于 0 的值。
3. 若 a 等于 b，则返回 0。
4. a 大于 b，则返回一个大于 0 的值。

Example:

replace(regexp,replacement) 方法用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串。

1. 接收两个参数；
2. 第一个参数代表的是被替换的元素
3. 第二个参数是要替换的新元素

说明：

字符串 stringObject 的 replace() 方法执行的是查找并替换的操作。它将在 stringObject 中查找与 regexp 相匹配的子字符串，然后用 replacement 来替换这些子串。如果 regexp 具有全局标志 g，那么 replace() 方法将替换所有匹配的子串。否则，它只替换第一个匹配子串。

replacement 可以是字符串，也可以是函数。如果它是字符串，那么没有匹配都将由字符串替换。但是 replacement 中的 $ 字符具有特定的含义。如下表所示，它说明从模式匹配得到的字符串将用于替换。
